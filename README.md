# LicenseKit

StoreKit 2 + license API client for Docked.

Part of [**Docked**](https://docked.my) — a floating, multi-display dock companion for macOS.

## Install

```swift
.package(url: "https://github.com/akhlaqahmad/docked-licensekit.git", branch: "main")
```

Then add `LicenseKit` to your target's dependencies.

## Platforms

See `Package.swift`. AppCore, DesignSystem, and LicenseKit support macOS 14+ and iOS 17+. DisplayKit, DockWindowKit, SettingsUI, and WorkspaceKit are macOS 14+ only.
